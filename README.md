# JCR DesWeb — WordPress Site Project

Proyecto de sitio web personal en WordPress con enfoque híbrido:

- portfolio profesional
- blog
- captación de clientes
- marca profesional
- laboratorio/documentación WordPress

## Objetivo
Construir una web profesional, escalable y documentada que sirva como activo comercial y como caso de estudio técnico.

## Stack actual
- WordPress
- Dinahosting
- Dominio: jcrdesweb.com

## Estado actual
Proyecto en fase inicial de infraestructura, documentación y versionado.

## Qué contiene este repositorio
- documentación del proyecto
- estructura base para código propio
- control de cambios por ramas
- futuras personalizaciones de tema/plugins
